## Як завантажити рахунок?

1. Відкрийте **Платежі** → натисніть **«+ Новий платіж»** або перетягніть файл у вікно
2. AI автоматично розпізнає дані з документа
3. Перевірте заповнені поля (отримувач, сума, IBAN, призначення)
4. Підтвердіть збереження

:::tip
Можна завантажувати кілька файлів одночасно — кожен буде оброблений окремо.
:::

## Які формати підтримуються?

Doc2Pay підтримує:

| Формат | Призначення |
|--------|-------------|
| **PDF** | Рахунки, акти, договори |
| **JPG / PNG** | Фото рахунків, скріншоти |
| **Excel** | Таблиці, звіти |

Детальніше — у статті [Формати файлів](/reference/file-formats/).

## Як експортувати в банк?

1. Виберіть потрібні платежі за допомогою чекбоксів
2. Натисніть **«Експорт»**
3. Завантажте згенерований CSV файл
4. Імпортуйте файл в Автоклієнт ПриватБанку або інший банківський клієнт

:::note
Формат CSV сумісний з Автоклієнтом ПриватБанку. Для інших банків можуть знадобитися додаткові налаштування.
:::

## Чи безпечно зберігати API ключі?

Так. Doc2Pay зберігає всі токени та API ключі з шифруванням **AES-256-GCM**. Доступ до них мають лише авторизовані користувачі вашого акаунту.

:::caution
Ніколи не передавайте токени стороннім особам. Якщо ключ скомпрометовано — негайно відкликайте його в банку та створіть новий.
:::
