## v0.5.1 — Поточна версія

**Дата:** березень 2025

Початковий реліз Doc2Pay з базовими можливостями:

- **Реєстрація та вхід** — створення акаунту, авторизація
- **Головна панель** — огляд компаній, платежів, активності
- **Компанії та групи** — управління ФОП та групами
- **Платежі** — завантаження рахунків, AI-розпізнавання, експорт у банк
- **Податки та зарплати** — базове планування
- **Баланс** — перегляд залишків по рахунках
- **Інтеграція ПриватБанк API** — отримання балансів, виписки, відправка платежів
- **Команда** — ролі та запрошення користувачів
- **Підписка** — тарифи Free, Pro, Enterprise

:::note
Детальний список змін у кожному релізі буде доповнюватися з оновленнями платформи.
:::
