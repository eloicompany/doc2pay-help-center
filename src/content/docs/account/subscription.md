Doc2Pay пропонує кілька тарифів для різних потреб бухгалтерів та ФОП.

## Тарифи

| Тариф | Ціна | Можливості |
|-------|------|-------------|
| **Free** | 0 грн/міс | До 10 рахунків на місяць |
| **Pro** | 499 грн/міс | Необмежено рахунків + всі інтеграції |
| **Enterprise** | Індивідуально | Для великих компаній, персональний підхід |

:::tip
На тарифі Pro доступні всі інтеграції: ПриватБанк, Monobank, Ощадбанк, ДПС Кабінет та інші.
:::

## Способи оплати

- **Картка** — Visa або Mastercard (рекурентні платежі)
- **Рахунок для юросіб** — оплата за рахунком для юридичних осіб

:::note
При оплаті рахунком термін активації Pro — до 3 робочих днів після надходження коштів.
:::

## Зміна тарифу

Перейти на вищий тариф можна в будь-який момент. Різниця буде перерахована пропорційно.
