Автоматичний розрахунок та контроль податків для всіх ФОП групи.

## Що показує розділ

- **Календар платежів** — що потрібно оплатити цього місяця
- **Суми** — ЄСВ, ЄП, ВЗ для кожного ФОП
- **Прогрес ліміту** — використання ліміту доходу для групи 3
- **Статуси** — оплачено / очікує / прострочено

## Ставки податків за групами

| Група ФОП | ЄСВ | Єдиний податок | Військовий збір |
|-----------|-----|----------------|-----------------|
| 1 група | 22% МЗП/міс | ~10% МЗП/міс (фікс.) | 10% МЗП/міс |
| 2 група | 22% МЗП/міс | ~20% МЗП/міс (фікс.) | 10% МЗП/міс |
| 3 група | 22% МЗП/міс | 5% доходу/квартал | 1% доходу/квартал |

## Налаштування реквізитів ДПС

1. Натисніть ⚙️ **«Налаштування»** біля ФОП
2. Оберіть **область** (регіон)
3. Введіть **IBAN** рахунків ДПС для ЄСВ та ЄП
4. Увімкніть **«Автосплата»** (необов'язково)
5. Натисніть **«Зберегти»**

:::note
IBAN для ЄСВ та ЄП можна вказати окремо. Автосплата вимагає [інтеграції з ПриватБанком](/integrations/privatbank/).
:::

## Оплата податку

1. Натисніть **«Оплатити»** біля податку
2. Оберіть спосіб:
   - **Через ПриватБанк API** — автоматична оплата
   - **Оплачено вручну** — позначити як оплачено
3. Підтвердіть

:::note
Для автоматичної оплати через API потрібно спочатку [налаштувати інтеграцію з ПриватБанком](/integrations/privatbank/).
:::
