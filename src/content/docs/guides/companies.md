Розділ **«Мої компанії»** — центральне місце для управління вашими суб'єктами господарювання.

## Структура

- **Групи** — об'єднують кілька ФОП/ТОВ (наприклад, «Клієнти Львів»)
- **ФОП/ТОВ** — окремі суб'єкти господарювання всередині груп

## Створення групи

1. Натисніть **«+ Нова група»**
2. Введіть назву групи
3. За потреби додайте опис (необов'язково)
4. Натисніть **«Створити»**

## Додавання ФОП/ТОВ

1. Натисніть **«+ Додати ФОП/ТОВ»**
2. Оберіть тип: **ФОП** або **ТОВ**
3. Заповніть поля:

| Поле | Опис | Обов'язкове |
|------|------|:-----------:|
| **Назва** | Повна назва (наприклад, ФОП Іванов І.І.) | ✅ |
| **Коротка назва** | Скорочена назва | — |
| **ЄДРПОУ/РНОКПП** | Код ЄДРПОУ або ІПН | ✅ |
| **Група ФОП** | 1, 2, 3 або 4 група | ✅ |
| **Платник ПДВ** | Так/Ні | ✅ |
| **IBAN** | Банківський рахунок | — |
| **МФО** | Код банку | — |
| **Назва банку** | Назва банку | — |
| **Телефон** | Контактний телефон | — |
| **Email** | Електронна пошта | — |
| **Адреса** | Юридична адреса | — |

4. Натисніть **«Зберегти»**

:::tip[Порада]
Можна додати **кілька банківських рахунків** для одного ФОП — натисніть «+ Додати рахунок».
:::

## Редагування та видалення

- **Редагування:** натисніть іконку редагування ✏️ біля ФОП/ТОВ або групи, внесіть зміни, натисніть **«Зберегти»**
- **Видалення:** натисніть іконку видалення 🗑️ та підтвердіть дію

:::caution[Увага]
Видалення групи **видалить усі ФОП/ТОВ** в ній разом з їхніми даними!
:::
