Розділ **Пошта** дозволяє підключити email-акаунт для автоматичного отримання рахунків. Система сама перевіряє вкладення, розпізнає дані та створює платежі.

## Як це працює

1. **Підключіть email** — через IMAP (Gmail, Outlook, корпоративна пошта)
2. **Система автоматично отримує** рахунки з вкладень листів
3. **AI розпізнає** дані з PDF, зображень, Excel — сума, реквізити, постачальник
4. **Створюються платежі** — ви лише підтверджуєте або редагуєте

:::tip[Порада]
Створіть окрему email-адресу спеціально для отримання рахунків, наприклад `invoices@yourdomain.com`. Так легше фільтрувати листи та уникати зайвого сміття.
:::

## Підтримувані формати

- PDF
- Зображення (JPG, PNG)
- Excel (XLS, XLSX)

## Налаштування IMAP

Для підключення знадобляться:

- Адреса IMAP-сервера
- Порт (зазвичай 993 для SSL)
- Логін та пароль (або пароль додатку для Gmail)

:::note[Важливо]
Для Gmail потрібно створити «Пароль додатку» в налаштуваннях безпеки — звичайний пароль не працює з IMAP через двофакторну автентифікацію.
:::
