---

Перегляд залишків на банківських рахунках усіх ФОП вашої групи.

## Що показує

| Елемент | Опис |
|---------|------|
| **Список рахунків** | IBAN та назва банку |
| **Поточний баланс** | Залишок на кожному рахунку |
| **Дата оновлення** | Коли останній раз оновлювалися дані |
| **Сортування** | За сумою або за назвою |

:::caution[Важливо]
Для отримання актуальних балансів потрібно налаштувати [ПриватБанк API](/integrations/privatbank/). Без інтеграції дані не оновлюватимуться автоматично.
:::
