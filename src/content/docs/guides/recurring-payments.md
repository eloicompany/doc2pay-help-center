---
title: Регулярні платежі
description: Шаблони повторюваних платежів з динамічними змінними.
---

Автоматизуйте платежі, що повторюються щодня, щотижня, щомісяця, щокварталу або щороку.

## Приклади

- Оренда офісу — щомісяця
- Мобільний зв'язок — щомісяця
- Хостинг — щороку
- Бухгалтерські послуги — щомісяця

## Створення шаблону

1. Натисніть **«+ Новий платіж»**
2. Заповніть:
   - **Назва** — назва платежу
   - **Отримувач** — IBAN та назва
   - **Сума** — розмір платежу
   - **Призначення платежу** — текст призначення
   - **Періодичність** — щодня / щотижня / щомісяця / щокварталу / щороку
   - **День виконання** — день виконання платежу
3. Натисніть **«Зберегти»**

:::note
Отримувач вказується через IBAN та назву. Для регулярних платежів змінні в призначенні оновлюються автоматично.
:::

## Динамічні змінні в призначенні

У полі «Призначення платежу» можна використовувати змінні:

| Змінна | Результат |
|--------|-----------|
| `{month}` | Поточний місяць (Січень, Лютий...) |
| `{prev_month}` | Попередній місяць |
| `{year}` | Поточний рік (2026) |
| `{quarter}` | Поточний квартал (Q1, Q2...) |

**Приклад:** `Оренда за {month} {year}` → «Оренда за Січень 2026»
