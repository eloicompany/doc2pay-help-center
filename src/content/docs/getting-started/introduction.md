**Doc2Pay** — хмарна платформа для бухгалтерів та ФОПів в Україні. Штучний інтелект автоматизує обробку рахунків-фактур і формує платіжні файли для банку.

## Для кого ця платформа

- **Бухгалтери** — які обслуговують кількох ФОП/ТОВ одночасно
- **ФОП** — які хочуть спростити роботу з рахунками та податками
- **Компанії** — де кілька людей працюють з фінансами

## Основні можливості

- **AI-розпізнавання рахунків** — завантажте PDF, фото або Excel, і штучний інтелект автоматично витягне дані
- **Платіжні файли для банку** — формування CSV для імпорту в Автоклієнт ПриватБанку та інших банків
- **Податки** — автоматичний розрахунок ЄСВ, ЄП, ВЗ з нагадуваннями про дедлайни
- **Зарплати** — управління виплатами працівникам
- **Регулярні платежі** — автоматизація повторюваних платежів
- **Баланси** — залишки на рахунках через API банку
- **Аналітика** — графіки доходів, витрат, категорій
- **Команда** — ролі та доступи для кількох користувачів
- **Інтеграції** — ПриватБанк API, Telegram, Email

## Системні вимоги

Doc2Pay — це веб-додаток, який працює в браузері. Потрібен сучасний браузер з підтримкою JavaScript:

- Google Chrome (рекомендовано)
- Mozilla Firefox
- Microsoft Edge
- Safari

:::tip[Порада]
Для найкращої роботи оновіть браузер до останньої версії.
:::
