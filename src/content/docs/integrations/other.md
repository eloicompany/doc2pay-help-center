Doc2Pay інтегрується з популярними сервісами для бухгалтерів та ФОП у Україні.

## Підтримувані інтеграції

| Інтеграція | Категорія | Опис |
|------------|-----------|------|
| **Monobank** | Банк | API для ФОП — баланси, виписки, платежі |
| **Ощадбанк** | Банк | API для ФОП — інтеграція з рахунками |
| **ДПС Кабінет** | Податкова | Електронний кабінет платника податків |
| **M.E.Doc** | Бухгалтерія | Обмін документами |
| **BAS** | Бухгалтерія | Бухгалтерські програми |
| **1С** | 1С:Підприємство | Інтеграція з 1С |
| **KeyCRM** | CRM | Система управління клієнтами |
| **KeepinCRM** | CRM | CRM для малого бізнесу |

:::note
Детальні інструкції з налаштування кожної інтеграції з'являться в оновленнях документації.
:::

## Категорії

- **Банки** — отримання балансів, виписок, відправка платежів
- **Податкова** — синхронізація з електронним кабінетом
- **Бухгалтерія** — обмін документами та даними
- **CRM** — інтеграція з системами управління клієнтами
