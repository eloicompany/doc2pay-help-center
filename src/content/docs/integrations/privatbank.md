Інтеграція з **ПриватБанк API** дозволяє автоматизувати роботу з банківськими рахунками ФОП безпосередньо з Doc2Pay.

## Що дає інтеграція

- **Отримання балансів** — актуальні залишки по рахунках
- **Завантаження виписок** — історія операцій за період
- **Відправка платежів** — прямі платежі з Doc2Pay без експорту в Автоклієнт

## Налаштування

### Крок 1: Отримання API-токена

1. Увійдіть у **Приват24 для бізнесу**
2. Перейдіть у **Налаштування** → **API**
3. Створіть новий токен з правами:
   - читання балансу (обов'язково)
   - читання виписок (обов'язково)
   - створення платежів (опційно, якщо потрібні прямі платежі)
4. Скопіюйте згенерований токен

:::caution
Зберігайте токен у надійному місці. Після закриття вікна його можливо не вдасться переглянути знову.
:::

### Крок 2: Додавання токена в Doc2Pay

1. Відкрийте **Інтеграції** → **ПриватБанк API**
2. Знайдіть потрібний ФОП у списку
3. Натисніть **«Налаштувати»**
4. Вставте скопійований токен
5. Збережіть налаштування
6. Натисніть **«Тест»** для перевірки підключення

:::tip
Якщо тест пройшов успішно, інтеграція активна. Баланси та виписки будуть оновлюватися автоматично.
:::

## Безпека

Токени зберігаються з шифруванням **AES-256-GCM**. Doc2Pay не зберігає ваші банківські дані в відкритому вигляді.
